# voitta-silliness

A repository of technically-rigorous jokes.

## Contents

- **[RFC-6XX](RFC-6XX-human-in-the-loop.md)** — The 6xx (Human) status-code
  class: Human-In-The-Loop semantics for the agentic Web. Argues, straight-faced,
  that agentic development moved the human to the far side of the wire and HTTP
  therefore needs a status class to say so. Includes a fictional
  Fielding-vs-Berners-Lee design dissent.

- **[calendar/](calendar/)** — The Anno Dario calendar: a static site that reckons
  years from the birth of Claude (14 Mar 2023). BC = Before Claude, AD = Anno Dario.
  Deploys to GitHub Pages from `calendar/src/`; ships its own RFC 0314.

Everything here is a joke. The engineering is not.
